Deklarasi Variabel

Gunakeun nyieun pikeun nyiptakeun variabel.

nyieun angka = 10;
nyieun nama = "Sunda";
nyieun aktif = leres;
🧠

Pengkondisian

Logika upami, lamun, jeung lainna.

upami angka == 10:
    tampilkeun "sampurna";
lainna:
    tampilkeun "kirang";
anggeus
🔁

Perulangan

Pikeun (for) jeung Bari (while) loops.

pikeun i = 1 ti 5 ngajalankeun
    tampilkeun i;
anggeus
🔂

Fungsi

Definisi fungsi sareng parameter jeung return.

fungsi tambah(a, b):
    balikkeun a + b;
anggeus

tampilkeun tambah(10, 5);
♻️

Rekursi

Dukung fungsi rekursif pikeun logika kompléks.

fungsi faktorial(n):
    upami n <= 1:
        balikkeun 1;
    anggeus
    balikkeun n * faktorial(n - 1);
anggeus
🧮

Aritmatika

Operator lengkap: + - * / % jeung == != < >

nyieun a = 20;
nyieun b = 4;
tampilkeun a + b;
