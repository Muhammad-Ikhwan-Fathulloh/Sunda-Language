Fitur Bahasa Sunda

🔢 Deklarasi Variabel

Gunakeun ngadeklarasikeun pikeun nyiptakeun variabel.

ngadeklarasikeun angka = 10;

🧠 Pengkondisian (Upami, Lamun, Lain Lamun)

Logika upami, lamun, jeung lain lamun pikeun percabangan logis.

upami angka == 10:
  tampilkeun "sampurna";
lamun angka == 9:
  tampilkeun "hampir";
lain lamun:
  tampilkeun "kirang";

🔁 Perulangan (Pikeun)

Pikeun ngalakukeun looping ti hiji angka ka nu sanesna.

pikeun i = 1 TI 5 ngajalankeun:
  tampilkeun i;

🔂 Fungsi Jeung Balikkeun

Ngadukung pembuatan fungsi ngagunakeun fungsi jeung balikkeun.

fungsi tambah(x, y):
  balikkeun x + y;

💬 Input Ti Pamaké

Ngadukung input interaktif ngagunakeun mangga_eusian.

ngadeklarasikeun nama = mangga_eusian("Asupkeun ngaran:");
tampilkeun nama;
