🌺 Sunda Language

Basa pamrograman saderhana nganggo basa Sunda pikeun diajar jeung ngalatih logika.

📦 GitHub Repositori

⚡ Cara Install

# Install via pip

pip install -e .


# Jalankeun program

sunda examples/hello_world.sunda


# Install VS Code syntax highlighting

Copy-Item -Recurse sunda-vscode "$env:USERPROFILE\.vscode\extensions\sunda-language"

Fitur Bahasa Sunda

🔢 Deklarasi Variabel

Gunakeun nyieun pikeun nyiptakeun variabel.

nyieun angka = 10;
nyieun nama = "Sunda";
nyieun aktif = leres;

🧠 Pengkondisian

Logika upami, lamun, jeung lainna.

upami angka == 10:
    tampilkeun "sampurna";
lainna:
    tampilkeun "kirang";
anggeus

🔁 Perulangan

Pikeun (for) jeung Bari (while) loops.

pikeun i = 1 ti 5 ngajalankeun
    tampilkeun i;
anggeus

🔂 Fungsi

Definisi fungsi sareng parameter jeung return.

fungsi tambah(a, b):
    balikkeun a + b;
anggeus

tampilkeun tambah(10, 5);

💬 Input Ti Pamaké

Input interaktif ngagunakeun tanya.

tanya nama;
tampilkeun nama;

🎨 VS Code Highlighting

Extension VS Code pikeun warna sintaks otomatis!

Copy sunda-vscode ka
~/.vscode/extensions/
